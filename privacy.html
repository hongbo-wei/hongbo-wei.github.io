Privacy Policy — Bookmark Sorter

Overview
Bookmark Sorter is a small browser extension that organizes your Chrome bookmarks locally. This Privacy Policy explains what (if any) data the extension accesses, how that data is used, and how to contact the developer.

Data accessed and purpose

  • Bookmarks: The extension requests the bookmarks permission only to read and reorder the bookmark tree in your browser so it can sort folders and bookmarks. All bookmark access and modifications occur locally in the user’s browser; no bookmark data (titles, URLs, structure) is uploaded, transmitted, or stored on any external server.
  • Notifications: The extension requests the notifications permission solely to display a completion or cancellation notice to the user. Notifications do not transmit bookmark data to third parties.
  • No other personal data: The extension does not collect analytics or other personal data.

Third parties and remote code

The extension does not call or use third‑party analytics, tracking, or telemetry services. All JavaScript and assets used by the extension are bundled in the extension package. The extension does not dynamically download or execute external scripts or WebAssembly at runtime.

Data retention and deletion

Because no bookmark data or other personal data is transmitted or stored remotely, there is nothing retained outside your browser. To stop the extension from accessing bookmarks, uninstall or disable the extension in Chrome.

Security

The extension limits its scope to the permissions declared in the manifest. All processing happens locally. If you have security concerns, review the extension code (background.js, popup.js) before installing.

Changes to this policy

We may update this policy to reflect changes to the extension. The latest version will be posted at the same URL.
